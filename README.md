By applying a student-specific transformation to raw sensor data, we simulate complex environmental noise and then reconstruct the statistical profile using the Method of Moments.1. MethodologyThe implementation is broken down into two distinct phases:Phase I: Non-Linear Data TransformationTo create a unique dataset, the original feature $x$ (Air Quality Index for $NO_2$) is mapped to a transformed variable $z$. This transformation introduces non-linearity based on parameters derived from the user's Roll Number ($r$).Transformation Formula:$$z = x + a_r \sin(b_r x)$$Parameter Logic:$a_r = 0.05 \times (r \pmod 7)$ (Amplitude of perturbation)$b_r = 0.3 \times (r \pmod 5 + 1)$ (Frequency of perturbation)Phase II: PDF Learning (Parameter Estimation)The distribution of the transformed variable $z$ is modeled using a Gaussian-style density function:$$\hat{p}(z) = c \cdot e^{-\lambda(z - \mu)^2}$$We "learn" the distribution by matching the moments of the dataset:$\mu$ (Location): The sample mean of the transformed data.$\lambda$ (Scale): Defined as $1/(2\sigma^2)$, where $\sigma^2$ is the sample variance.$c$ (Normalization): Ensures the integral of the PDF equals 1, calculated as $\frac{1}{\sigma\sqrt{2\pi}}$.2.
